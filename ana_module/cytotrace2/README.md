| 参数               | 是否为必须参数 | 默认值 | 描述                                            |
| ------------------ | -------------- | ------ | ----------------------------------------------- |
| `--rds`            | 是             | 无     | seurat对象的rds路径                             |
| `--outdir`         | 是             | 无     | 结果输出路径                                    |
| `--celltype_col`   | 是             | 无     | 细胞类型所在列的列名，必须在rds@meta.data中存在 |
| `--spec`           | 否             | human  | 支持human或mouse                                |
| `--reduction_name` | 否             | umap   | 使用哪个降维的结果                              |
