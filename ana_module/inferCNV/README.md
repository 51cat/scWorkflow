| 参数                    | 是否为必须参数 | 默认值                              | 描述                                                      |

| `--rds`                 | 是             | 无                                  | seurat对象的rds路径                                       |

| `--exclude_group`       | 否             | 无                                  | 哪些组不用于分析                                          |
| `--posfile`             | 否             | refdata_gex_GRCh38_2024_A_human_pos | 位点信息文件                                              |
