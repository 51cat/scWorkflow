| 参数        | 是否为必须参数 | 默认值        | 描述                              |
| ----------- | -------------- | ------------- | --------------------------------- |
| `--rds`     | 是             | 无            | seurat对象的rds路径               |
| `--outfile` | 是             | 无            | 结果文件输出路径                  |
| `--pk`      | 可选参数       | 1:20          | 主成分数，例如 `1:10`             |
| `--pN`      | 可选参数       | 0.25          | DoubletFinder 参数 pN             |
| `--method`  | 可选参数       | DoubletFinder | 双细胞检测方法, 只有DoubletFinder |
