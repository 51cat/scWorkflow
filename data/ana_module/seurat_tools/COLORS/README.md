| 参数           | 是否为必须参数 | 默认值 | 描述                                                   |
| -------------- | -------------- | ------ | ------------------------------------------------------ |
| `--colors`     | 是             | 无     | 需要添加的颜色，逗号分隔                               |
| `--name`       | 是             | 无     | 颜色在数据库中的名称                                   |
| `--color_type` | 可选参数       | d      | 值为c或d，d代表离散变量的颜色类型，c为连续变量颜色类型 |
