| 参数               | 是否为必须参数 | 默认值   | 描述                                        |
| ------------------ | -------------- | -------- | ------------------------------------------- |
| `--rds`            | 是             | 无       | seurat对象的rds路径                         |
| `--outdir`         | 是             | 无       | 结果输出路径                                |
| `--gene_list`      | 是             | 无       | 包含一列基因信息的文本文件, 列名是gene      |
| `--group_col`      | 是             | 无       | 分组所在列的列名，必须在rds@meta.data中存在 |
| `--reduction_name` | 可选参数       | 可选参数 | 使用哪个降维结果                            |
